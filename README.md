# Palm Beach Influence 100

A premium Next.js + Tailwind website for Palm Beach Influence 100, a curated business recognition and media platform focused on influential Palm Beach companies, institutions, and platforms.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open `http://localhost:3000`

## Project structure

- `app/` - Next.js routes and pages
- `components/` - reusable UI components
- `app/globals.css` - Tailwind base styles
- `tailwind.config.ts` - Tailwind configuration
